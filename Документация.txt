Приложение включает следующие функции:

    Добавление заметки: Позволяет пользователю ввести текст заметки и сохранить её.
    Просмотр заметок: Отображает список всех сохранённых заметок.
    Удаление заметки: Удаляет выбранную пользователем заметку.
    Поиск заметки: Выполняет поиск заметок по ключевым словам.
    Выход из приложения: Закрывает приложение.

Использование приложения:
- Добавление заметки

    Выберите "1" из главного меню.
    Введите текст заметки.
    Сохраните заметку.

- Просмотр всех заметок

    Выберите "2" для отображения всех заметок.
    Просмотрите список заметок на экране.

- Удаление заметки

    Выберите "3" для удаления заметки.
    Введите номер заметки, которую хотите удалить.

- Поиск заметки по ключевому слову

    Выберите "4" для поиска заметок.
    Введите ключевое слово для поиска.
    Просмотрите результаты поиска.

Технологии
 - .NET Core 8.0
 - Библиотеку: Microsoft.Data.Sqlite
